# 🔐 Spring Security Basic Auth Example

Проект демонстрирует базовую аутентификацию с помощью **Spring Security** и хранение пользователей в базе данных **PostgreSQL** с использованием **JDBC**.  

---

## 🚀 Технологии
- ☕ **Java 21**
- 🌱 **Spring Boot / Spring Security**
- 🗄 **PostgreSQL**
- 🔌 **JDBC**
- 🐳 **Docker & Docker Compose**
